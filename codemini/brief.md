# brief.md — codemini.ru

## Что это

Партнёрский сайт под офферы САЛИД в нише детского IT-образования.
Продвигает курсы программирования, робототехники и IT для детей 5–17 лет.
Монетизация: партнёрские ссылки через САЛИД (CPS, CPL).

## Сайт

- Домен: codemini.ru
- Хостинг: Beget
- CMS: WordPress
- Партнёрская сеть: САЛИД (salid.ru)

## Целевая аудитория

Родители детей 5–17 лет, которые хотят:
- Дать ребёнку перспективную профессию
- Чем-то занять ребёнка после школы
- Найти онлайн-курс по программированию/IT
- Подготовить к ЕГЭ по информатике
