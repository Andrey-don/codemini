Программирование
7–17 лет

Самое популярное направление. Дети учатся писать код на Scratch, Python или JavaScript — создают игры, сайты и приложения.

Чему научится ребёнок
  • Писать программы на Python или Scratch
  • Создавать игры и веб-страницы
  • Думать алгоритмически и решать задачи
  • Делать проекты для портфолио

Робототехника
5–14 лет

Ребёнок собирает и программирует роботов — видит результат сразу. Развивает инженерное мышление, пространственное восприятие и интерес к технике.

Чему научится ребёнок
  • Собирать роботов из конструктора
  • Программировать движение и датчики
  • Решать инженерные задачи
  • Участвовать в олимпиадах по робототехнике

Разработка игр
7–17 лет

Дети создают игры в Roblox, Minecraft, Scratch или Unity. Лучший способ мотивировать ребёнка — дать ему создать то, во что он сам играет.

Чему научится ребёнок
  • Создавать игровые уровни и персонажей
  • Программировать игровую логику
  • Основам дизайна игрового процесса
  • Делиться играми с друзьями

Веб-разработка
10–17 лет

HTML, CSS, JavaScript — ребёнок создаёт настоящие сайты. Одно из самых востребованных направлений для старшеклассников с прицелом на профессию.

Чему научится ребёнок

  • Добавлять интерактивность на JavaScript
  • Создавать полноценные сайты
  • Публиковать проекты в интернете

Для детей, которым нравится рисовать и создавать — учат графическому и UI/UX дизайну. Хорошо сочетается с веб-разработкой.
