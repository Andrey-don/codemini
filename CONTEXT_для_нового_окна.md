## Важные нюансы САЛИД

- Атрибуция — **последний клик**. Наша ссылка должна быть последней перед регистрацией.
- **Сохранять скрины рекламных объявлений** для EasyDesi, Rebotica, Айтигенио, Рокет Тех — условие выплаты.
- **Айтигенио:** регистрация засчитывается только при вводе имя+пол+возраст+пароль. Предупреждать пользователя ДО перехода.
- **Рокет Тех промокод PODAROK** — скидка 50% на первый месяц. Эксклюзив, недоступен ИИ.
- Не покупать курсы по своей ссылке. Не накручивать трафик.
- Запрещённый трафик у всех: Instagram/FB таргет, мотивированный, спам, реклама на бренд.

## Блок результата квиза (UX требования)

Каждый результат должен содержать:
1. Название школы + краткое объяснение почему подходит
2. Ключевые факты: возраст, формат, цена от...
3. Для Рокет Тех — промокод **PODAROK** жирным
4. Кнопка «Записаться на бесплатный урок» с партнёрской ссылкой
5. Текст перед кнопкой: «После клика заполните короткую форму на сайте школы (имя, email, телефон) — это займёт 1 минуту»
6. Прогресс-бар (шаг 1 из 3, шаг 2 из 3...)
7. Кнопки выбора, не выпадающие списки
8. Мобильная адаптация обязательна
